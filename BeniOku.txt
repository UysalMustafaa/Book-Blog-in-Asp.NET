                 
<----- SQL' De Açma ------>

4-Bilgisayarımız da kurulu olan Microsoft Sql Server'ın dosyalarına giriyoruz. Şu adımları takip ediyoruz;
5-MSSQL.SQLEXPRESS isimli dosyaya giriyoruz.
6-Orada olan Backup dosyasına KitapBlog.bak adlı dosyayı kopyalıyoruz.
7-Sql Server Managment  studio 2012 veya daha üst versiyonunu açıyoruz.
8-Sol taraftaki Object Explorer penceresinde Database üzerinde sağ tıklıyoruz ve Restore Database’ i seçiyoruz.
9-Açılan pencerede Device seçeneğine tıklıyor ve açılan pencereden Add diyerek istediğimiz bak dosyasını seçiyoruz.
10-Hemen ardından altındaki Database kısmından açmış olduğumuz .bak dosyasının yani veritabanımızın adını seçiyoruz.
 En son olarak aşağıdan kırmızı işaratli alanda Restore’ nin altında kalan yerdeki kutucuğa tıklayarak Ok diyoruz.

<----- Visual Studio'da Açma ----->

11-Visual Studio'yu açıp sol üst taraftan "File" 'e tıklıyoruz.
12-File sekmesinden "Open" Seçeneğini seçip kendi " Website" 'mizi açıyoruz.
13-Sağ Tarafta olan Solution Explorer adlı sekmeden "Anasayfa.aspx" çalıştırdıktan sonra site packages kendiliğinden yükleyecektir.
14-Yükledikten sonra Anasayfa.aspx açılacaktır ve site kullanıma hazırdır.
